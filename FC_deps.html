Known issues:

  • The most recent version of gcc in Fedora's 30 and Rawhide repositories is actually 9.1.1. However, Repology's system has marked this internally as "ignored" for an undisclosed reason. What's shown above are the next most-recent versions of gcc available from those repositories.
  • The libGL library used to be provided by mesa. Now it is provided by libglvnd. Not really an issue as such, just an observation.
    • That said, I'm not sure what provides libGL on (Open|Free)BSD as the package lists for both fail to identify a package that provides that library.
